# Computer Graphics

A collection of graphics programming labs developed using Node.js and various Web Graphics standards.

## Technologies Explored
* **Three.js:** High-level 3D animations and scene management.
* **Canvas 2D:** Low-level pixel manipulation and 2D procedural drawing.
* **SVG & X3D:** Vector-based graphics and declarative 3D modeling.
